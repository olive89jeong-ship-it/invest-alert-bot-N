# Invest Alert Bot - Theme Radar

이 버전은 개별 종목 가격 체크가 아니라, 전반적인 시장 흐름에서 어떤 테마가 뜨고 있는지 감지하는 텔레그램 알림 봇입니다.

## 핵심 목적

기존:
```text
GRID 얼마
PAVE 얼마
ETN 얼마
```

변경:
```text
오늘 시장에서 강한 테마는 무엇인가?
뉴스와 가격이 동시에 움직이는 테마는 무엇인가?
아직 가격은 약하지만 뉴스가 늘어나는 조기 테마는 무엇인가?
```

## 출력 구조

텔레그램에는 아래 형식으로 발송됩니다.

```text
🤖 시장 테마 레이더

1. 오늘의 결론
- 강한 테마
- 조기 감지 테마
- 약한 테마

2. 테마 랭킹
- AI 전력 인프라
- 전력망·스마트그리드
- 미국 인프라
- 전력기기
- 송전망 시공
- 전력망 운영

3. 왜 떴는지
- 관련 뉴스 제목
- 관련 ETF/종목 흐름

4. 참고 종목 설명
- GRID가 뭔지
- PAVE가 뭔지
- 숫자가 뭘 의미하는지
```

## GitHub Secrets

아래 3개가 필요합니다.

```text
OPENAI_API_KEY
TELEGRAM_BOT_TOKEN
TELEGRAM_CHAT_ID
```

## 실행

```text
Actions
→ Invest Alert Bot - Theme Radar
→ Run workflow
```
